git init > cria um repo git na pasta atual
git status > mostra o status do trabalho (de cada arquivo)
git add nome_arquivo > adiciona arquivo para ser commitado
git commit -m "msg" > faz commit do que estava para ser commitado
git log > Mostra todo o histórico de commit
git diff <nome do arquivo> > Mostra as alterações que foram feitas no arquivo.
git reset <nome do arquivo> > Volta o arquivo adicionado anteriormente no comando (git add) para o estado anterior.
git checkout -b <Nome da branche> > Cria uma nova branche com a cópia dos comites anteriores
git checkout <Nome da branche> > navega entre branche
git branche > sinaliza (em verde) em qual branche está
git merge <nome da branche> copia arquivos da branche referenciada
git branch -d <nome da branch> > deleta uma branche *obs. Eu não posso está logado na branch que irei excluir.
Git push origin <branch: main> > Envia/Salva as últimas alterações do arquivo no GitHub 

## FLUXO DE TRABALHO
1. Criar o diretório (pasta) do projeto
2. Iniciar o git dentro do diretório criado (git init .)
3. Edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. Volta para o passo 3.
